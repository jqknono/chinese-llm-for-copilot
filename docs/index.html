插件简介与推荐

Coding Plans for Copilot 是一个 VS Code 扩展，帮助你把不同厂商的编码套餐接入 Copilot Chat，统一在编辑器中切换供应商与模型。

  • 推荐给希望在 VS Code 内统一管理多家编码套餐的开发者。
  • 推荐给需要低成本试用不同模型、快速对比体验的团队。
  • 推荐优先从 VS Code 插件市场安装，更新与维护更方便。
